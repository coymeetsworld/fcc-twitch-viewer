About Twitch TV Viewer

The Twitch TV Viewer was written by Coy Sanders to complete one of the fourIntermediate Front-End Development Projects for FreeCodeCamp to earn the Front-End Development Certification.

This application makes several Twitch API calls to gather information on featured streams as well as a hard-coded list of users. Documentation on the Twitch APIs can be found here.
